# 01 · 环境搭建

## 你需要什么

这个项目是纯 Python 脚本，没有任何图形界面。你需要：

| 工具 | 用途 | 安装 |
|------|------|------|
| Python 3.10+ | 运行脚本 | [python.org](https://python.org) |
| Git（可选） | 版本管理 | [git-scm.com](https://git-scm.com) |
| VS Code（推荐） | 编辑器 | [code.visualstudio.com](https://code.visualstudio.com) |

## 第一步：安装依赖

项目根目录有一个 `requirements.txt`：

```
feedparser==6.0.11
requests==2.32.4
openai==1.82.0
python-dotenv==1.1.0
```

在终端执行：

```bash
pip install -r requirements.txt
```

- `feedparser` — 解析 RSS/XML 订阅源
- `requests` — 发 HTTP 请求（抓网页、调 API）
- `openai` — 调用大模型（OpenAI 兼容接口）
- `python-dotenv` — 从 `.env` 文件加载环境变量

## 第二步：配置 `.env`

复制 `.env.example` 为 `.env`，填入你的密钥：

```bash
# OpenRouter API 密钥（调用 Kimi K2.6 模型）
OPENROUTER_API_KEY=sk-or-v1-你的密钥

# Notion 集成 Token
NOTION_TOKEN=ntn_你的Token

# Notion 数据库 ID
NOTION_DATABASE_ID=你的数据库32位ID

# Server酱推送密钥（可选，不填就跳过手机推送）
PUSH_KEY=你的SendKey
```

> [!tip] `.env` 在 `.gitignore` 中，不会被提交到 GitHub
> 密钥泄漏的后果：别人可以用你的额度调用 API、访问你的 Notion

## 第三步：跑起来

```bash
cd AI-Daily-News
python main.py
```

如果一切正常，你会看到：
```
[INFO] __main__: [1/3] Fetching news...
[INFO] __main__: [2/3] Generating report...
[INFO] __main__: [3/3] Pushing to Notion...
```
